# metadata

> Scrape metadata from websites.Inspired by [one-tab-group/metafy](https://github.com/one-tab-group/metafy).

## Usage

Enter a valid `$URL` as params

```bash
curl https://metadata-lilac.vercel.app/api?url=$URL
```
